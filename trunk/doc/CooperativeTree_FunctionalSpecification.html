Cooperative Tree

Functional Spec

The Cooperative Tree Project aim to develop an organization helper application around a shared tree data structure. It can be usefull for event scheduling (movies, restaurant, party) and project managing.
The application is based on a client server model.
Users will interact with each other though a shared data structure.


1. Each user can propose ideas to other user by creating or adding them as "Idea nodes" into a tree data structure .
2. An idea node can be refined by appending sub idea nodes   to it.
4. An idea node can be appended to many other idea nodes. It allow to not duplicate information when needed. It implies that an IdeaNode can't has itself has child or sub Child. For next version
5. If a user agrees an other user's idea-node he can vote for it , each vote has the same weight .
7. User can propose counter idea-nodes for a given idea-node. A new "competitive node" then replace the first idea . This competitive node then contains the first and all the counter ideas and the counter idea.
8. To find out that as been decide so far, a validation policy as to be choosen. This validation policy will be used to compute if an idea node will be present in the final validated tree .
9. In the default validation policy , only idea node with at least one vote is validated. For idea nodes belonging to a competitive node , only the idea node with the highest vote count is validated . if two  idea node have the same vote count, none of the idea nodes are validated.
10. An idea node contains :
    - a title name (short length), that is display in the tree, an represents briefly the meaning.
    - a description (long length), that is display as a pop-up to have a accurate understanding of the meaning.
    - the reference of the creator of the idea node.
    - the reference of all user who votes this idea.
    - the total count of votes for this idea-node


This is this the interface method that user can use :

- Node appendNew(Node parent)
- Node insertNewBefore(Node parent,Node Brother)
- remove(Node)
- setName(Node node,String name)
- setCreator(Node node,String creator)
- setDescription(Node node,String description)
- vote(Node node,String voter)
- Node appendCompetitive(Node parent,Node targetNode)
- Node insertCompetitiveBefore(Node parent,Node targetNode)
